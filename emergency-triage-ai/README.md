# Emergency Triage AI

This project is a simple AI-based emergency response triage assistant that analyzes user messages and classifies them based on urgency.

## How it works
The system checks the input text for emergency-related keywords such as "help", "accident", "fire", etc. If any of these keywords are detected, the message is classified as HIGH PRIORITY. Otherwise, it is marked as LOW PRIORITY.

## Features
- Detects emergency situations
- Classifies messages into priority levels
- Simple and fast processing

## Tech Stack
- Python
- Basic Natural Language Processing (NLP)

## Usage
Run the app.py file and enter a message. The system will analyze and display the priority level.
